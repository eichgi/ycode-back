## Ycode Backend

The following are the steps you need to run so you can serve and consume the service.

- Set up the MySQL database connection on .env
- Run:
```
php artisan migrate:fresh --seed
```
- Once the database is provisioned and seeded you have to serve it with:
```
php artisan serve --host=localhost
```  
This should start a server on http://localhost:8000

- Please go to the Ycode Frontend
